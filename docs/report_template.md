# Report template

Use this template when interpreting generated results.

## Research question

Can explainable and fairness-aware recommender systems reduce popularity bias, filter bubbles, and exposure inequality while preserving relevance and improving user trust?

## Dataset statement

State whether the run used synthetic data or reviewed real data under governance approval.

## Method comparison

Summarize relevance, exposure, diversity, filter-bubble, explanation, and trust proxy metrics for each method.

## Fairness findings

Identify under-exposed provider groups, user subgroup gaps, and whether reranking reduced exposure concentration.

## Filter-bubble findings

Review the highest-risk users and check whether category/provider entropy is too low.

## Human review notes

Document tradeoffs and any limitations before considering real-world use.
